Advanced Installation

This page walks you through installing everything manually. It uses Mod Organizer 2 and covers all required mods step by step. If you want the automated version, use the Wabbajack install instead.
This version of the guide assumes that you have relatively strong computer and troubleshooting skills.

Prerequisites

While not all of these are necessarily required, I highly recommend that you install Visual C++ Runtime Libraries and update your graphics drivers, especially if you haven't in a while or haven't ever.

A Good Archiving Tool

This is a tool for extracting and manipulating .zip, .rar, .7z and other compressed files.

  • I recommend NanaZip for Windows 11 users.
  • I recommend 7-Zip for Windows 10 users.

Visual C++ Runtime Libraries

This is a compilation of old and new C++ runtime libraries. Installing these libraries fixes some issues with running the game on modern systems. A lot of older games need these as well, so it doesn't hurt to have them installed.

  1. Download the file from Techpowerup.
  2. Extract the archive using your archiving tool of choice.
  3. Open the folder.
  4. Right click install_all.bat and run as administrator.
  5. Work your way through each install prompt until finished.

Graphics Drivers

Ensure that your graphics drivers are up to date.

  • Nvidia Graphics Drivers can be downloaded through the NVIDIA App.
  • AMD Graphics Drivers can be downloaded through AMD Adrenalin.
  • Intel Arc Graphics Drivers can be downloaded through Intel Arc.

Nexus Mods Account

This is a hard requirement for installing mods from Nexus. If you do not already have a nexus mods account, you will need one. You can create an account here.

Mod Organizer 2

Mod Organizer 2 (MO2) is an open source mod manager. If you've ever modded Bethesda games you'll likely recognize it. MO2 doesn't install mods directly into the games folder but instead creates a virtualized environment that is added to the game folder at runtime.


Benefits include:

  • Safe mod installation and uninstallation
  • The ability to modify mod priorities and overwrites at any time
  • Multiple mod profiles
  • Non destructive mod management keeps the game install clean

Installing Mod Organizer 2

  1. Download the latest Mod Organizer 2 .
  2. Once the download has finished, run the installer.
  3. Accept the license and click Next.
  4. When asked about the install location, use the default or select what best suits you, I do as follows.
  5. Click Next and select Recommended Install.
  6. On the Start Menu folder page, keep the default name or set your preferred name.
  7. Decide whether you want a desktop shortcut.
  8. Click Next, then click Install.
  9. Once the install is complete, ensure that Launch Mod Organizer is ticked, then click Finish.

Creating a New Instance

  1. Click next
  2. Select "Create a portable instance"
  3. Select "The Witcher Enhanced Edition"
  4. Use the defaults, keep automatic archive invalidation checked
  5. Click next
  6. Default Location should look the same as MO2's installation if you selected a portable instance
  7. Click next
  8. Click finish
  9. Choose whether you'd like to see the tutorial
  10. Close the wiki popup
  11. Select Import Nexus Categories

Nexus Mods MO2 Integration

  1. You should be asked to set up Nexus mod integration, "nxm links", select Yes
  2. Sign in, if necessary, and authorize MO2 integration with Nexus Mods

Installing Witcher Mods in MO2

Understanding Nexus

  1. When linked to a mod, select the "Files" tab.

    When the guide says to install a mod, it will be formatted as "Main File - Example 1.5", this is referring to the mod's category on Nexus and the file name, unless otherwise specified.

  2. Download the indicated file(s) using the mod manager download button.
    Note: Some files may only have a manual download option and you will need to import the archive manually.
  3. Open the Downloads tab in MO2
  4. Double-click on the mod to install it
  5. Most Witcher mods will need some extra work to be formatted
  6. Most Mods should ALWAYS have this structure in order to make sure other mods files are being properly overwritten and so that multiple of the same file are not loading at the same time. Raw mod files refer to the actual texture/script files themselves. data > Override > “raw” mod files

Example Mod Installation

  1. Enter the mod manager downloads tab
  2. Double-click the mod inside the downloads panel, this will start the installation process and tell you the mod is not formatted in the expected way
  3. In this case we only want The Witcher Texture overhaul, an easy way to eliminate all other options is to right click “The Witcher Texture Overhaul” and select “Set as <data directory>”
  4. This will leave all the loose files inside the data directory, however, we need them inside an Override folder. Right click <data> and select “Create directory…”, name it Override
  5. Move all the loose texture files inside the Override folder, an easy way to do this is to select the top file, scroll to the bottom Shift + Click the bottom file and this will select all files, move them to Override. Your final data structure should look like this.
  6. Click OK

Installing DXVK

Installing Reshade

  1. Download the latest Reshade WITH FULL ADD-ON SUPPORT
  2. Run the setup tool
  3. Find and select Witcher.exe inside the system folder of your game. Typically:
    "Steam\steamapps\common\The Witcher Enhanced Edition\Data\Override"
  4. Select Vulkan as the rendering api, since we're using dxvk to convert d3d9 commands to vulkan
  5. Checkmark the following packs
    • SweetFX by Ceejay.dk
    • Legacy effects
    • FXShaders by luluco250
    • qUINT by Marty McFly
    • Color effects by prod80
    • dh-reshade-shaders by AlucardDH
    • Insane-Shaders by Lord of Lunacy
    • AstrayFX
    • iMMERSE by Marty Mcfly
  6. Click run inside of MO2, from now on only launch the game through MO2. Mods will only load if the game is run through MO2
  7. Once loaded into the game, hit "home" on your keyboard to open reshade.
  8. Click Skip Tutorial
  9. Select the dropdown at the top, scroll down until you find WitcherReshade.ini, select the file and click Select
  10. Select Performance mode at the bottom of the page. This will fully compile and lock in settings, netting you more performance. If you want to change specific settings in filters you can turn it off and on at your will.
  11. Click on the add-ons tab
  12. Enable "Copy depth buffer during frame to prevent artifacts", this should also enable "Copy depth buffer before clear operations". This will allow reshade to access the depth buffer and inject ambient occlusion and other effects.
  13. That's it! Your game should now be modded and playable as intended. You can continue to add some more overhaul mods that couldn't be included in Wabbajack or continue playing!
