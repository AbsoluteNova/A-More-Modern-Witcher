Wabbajack

Wabbajack is an automated installer for the mods and MO2. This install allows for minimal user intervention, basically, you don't have to do as much work.

Installation

  1. Download the Lastest Wabbajack
  2. Download my Wabbajack file by clicking "A.More.Modern.Witcher.wabbajack"
  3. Create a new folder anywhere outside of any default Windows folders, and the game folder itself. I recommend making a Modding folder in your root directory and a Wabbajack folder inside of it.

    4. "C:\Modding\Wabbajack"

  4. Place the downloaded "Wabbajack.exe" and "A.More.Modern.Witcher.wabbajack" in this folder
  5. Run "Wabbajack.exe"
  6. Select Install From Disk
  7. Where it says "Target Modlist", click the three dots, then locate and select "A.More.Modern.Witcher.wabbajack" inside our "Modding\Wabbajack" folder
  8. Create a new folder inside "Modding", call it "A More Modern Witcher"

    9. "C:\Modding\A More Modern Witcher"

  9. Where it says "Modlist Installation Location", click the three dots and select the folder:

    10. "C:\Modding\A More Modern Witcher"

  10. Create a new folder inside "Modding" called "Downloads"

    11. "C:\Modding\Downloads"

  11. Where it says "Resource Download Location", click the three dots, and select the folder:

    12. "C:\Modding\Downloads"

  12. Wabbajack should look similar to this:
  13. Begin the installation by clicking the green arrow.
  14. Accept the Nexus Mods API request
  15. If you are not a Premium user you will need to manually click download for each mod.
  16. Once complete, the installation will show a green Installation Complete screen, you are safe to close Wabbajack.
  17. If you see a red Installation Failed screen, try logging in through Wabbajack's settings, then reinstall the list to the same folder.

Running The Game

This mod guide uses the MO2 mod manager to streamline the installation process for installing mods and to make your life easier for launching the game and disabling anything you don't want.

Post-Install Steps

Installing Reshade

  1. Download the lastest Reshade WITH FULL ADD-ON SUPPORT
  2. Run the setup tool
  3. Find and select Witcher.exe inside the system folder of your game. Typically:
    "Steam\steamapps\common\The Witcher Enhanced Edition\Data\Override"
  4. Select Vulkan as the rendering api, since we're using dxvk to convert d3d9 commands to vulkan
  5. Checkmark the following packs
    • SweetFX by Ceejay.dk
    • Legacy effects
    • FXShaders by luluco250
    • qUINT by Marty McFly
    • Color effects by prod80
    • dh-reshade-shaders by AlucardDH
    • Insane-Shaders by Lord of Lunacy
    • AstrayFX
    • iMMERSE by Marty Mcfly
  6. Click run inside of MO2, from now on only launch the game through MO2. Mods will only load through MO2
  7. Once loaded into the game, hit "home" on your keyboard to open reshade.
  8. Click Skip Tutorial
  9. Select the dropdown at the top, scroll down until you find WitcherReshade.ini, select the file and click Select
  10. Select Performance mode at the bottom of the page. This will fully compile and lock in settings, netting you more performance. If you want to change specific settings in filters you can turn it off and on at your will.
  11. Click on the add-ons tab
  12. Enable "Copy depth buffer during frame to prevent artifacts", this should also enable "Copy depth buffer before clear operations". This will allow reshade to access the depth buffer and inject ambient occlusion and other effects.
  13. That's it! Your game should now be modded and playable as intended. You can continue to add some more overhaul mods that couldn't be included in Wabbajack or continue playing!

Additional Mods - Manual Installation

There are some mods I couldn't include in the WabbaJack installation due to them being hosted on Moddb, so to get the full experience you'll have to install these manually yourself. Fortunately I've left a guide here to do so.

    Scabbard Mod
  1. Click the big red "Download Now" button
  2. This mod is packaged in an exe file, so we need to handle it differently than we would with a typical loose file mod.
  3. Create a file called "scabbards mod" somewhere easy to find, like your desktop.
  4. Execute the file, when it asks for your witcher game location point it to the folder you just made instead.
  5. Select "The Witcher EE" when it presents you with a dropdown.
  6. Finish the download and you should be presented with this in your scabbards mod folder
  7. Right click Data, Select send to and then "Compressed (zipped) folder"
  8. Name it "Scabbards mod"
  9. Install "Scabbards mod.zip" into MO2
  10. Right click DATA and select "set as directory", you should be given the following.
  11. Click OK
